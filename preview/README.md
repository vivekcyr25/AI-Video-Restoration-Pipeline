# Preview Site

Interactive scroll-driven demo for the AI Video Restoration Pipeline.

**Live:** https://vivekcyr25.github.io/AI-Video-Restoration-Pipeline/

```bash
npm install
npm run dev
```

Built with React, Tailwind CSS, and WebGL shaders.

## Build

```bash
npm run build
```

## Audio

Background music plays at 30% volume. Use the speaker button to mute/unmute.

## Deploy

Changes under `preview/` are published through the GitHub Pages workflow on `main`.
